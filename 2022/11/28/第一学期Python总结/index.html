变量

定义变量实质是将对象地址与变量绑定

变量名是内存地址的别名，即指向对象的引用

变量本身无类型，它指向的对象有类型

命名规则
  1. 第一个字符必须是字母或下划线，尽量小写，尽量不用下划线
  2. 标识符其他部分由字母、数字、下划线组成
  3. 大小写敏感
  4. 不能与关键字相同
  5. 类名使用驼峰命（CamelCase）名风格，首单词小写,其他单词大写、私有类可以用下划线
  6. 私有函数可用下划线开头
  7. 变量名尽量采用小写，多个单词用下划线隔开
  8. 常量全用全大写，多个单词用下划线隔开

字典

特点

  • 使用花括号{} ，元素由键和值两部分组成 {值：键}

  • 无序，键不可重复，可嵌套，可枚举

  • 见必须为不可变类型：底层实现为哈希技术

  • 字典对象本身是可变对象，元素（即键值对）可增删

常用操作

  • 键访问

    • 读访问，根据键获取

    • 写访问：已经存在这修改对应值，否则新增键值对

    • del操作：根据删除指定键对

  • 键对是否存在判断：in，字典默认为键集合

  • formkeys(iter,default):批量创建键值对，参数iter表示键集合，default是统一的默认值

  • keys(),values(),items():分别获取键集，值集、元素集 获取和要转型才可用哟

  • get(key,default):获取key对应的值，不存在则返回默认值default 常用

  • setdefault(key,default):和get相似，区别是不存在将在字典中新增键值对

集合类型

特点

  • 无序，不可重复，可枚举，不可嵌套

  • 使用花括号（空字典用“{}“表示，空集合用set()的方式）

  • 集合对象本身可修改，但集合内的元素要求不可变

  • 集合底层实现也使用了哈希技术，集合可以被理解为只有键的字典

    • 子集/超集：<= or issubset函数， >= or issuperset函数

  • 添加元素：add函数

  • 删除元素：remove函数

  • 是否相交判断：isdisjoint函数，不相交为True，相交为False

参数传递方式

  • 位置参数：按传递顺序匹配

  • 命名参数（关键字参数）：根据关键字匹配

  • 参数默认值：Python实现函数重载的方式

  • 规则1：命名参数只能位于位置参数之后

  • 规则2：定义时，默认参数必须再最后

  • 默认值示例：定义生成有理数字符串的函数，常见应该包含分子、分母两个参数；但整数也是有理数，是分母为1的分数。对这种现象的表达，其他语言可以使用函数重载（同名方法，参数不）的方式进行模拟，但python不支持函数重载，多个函数有效的是最后一个函数定义。

  • 可变参数：Python支持参数个数不确定的情况

    • 可变位置参数
    • 可变命名函数
    • 组合类型解包

  • 全适应参数：（*args, **keyargs):

  • 限制命名参数：第一个参数位置使用*，则可强制后面参数必须命名

文件操作

文件概述

  • 文件分类：

    • 文本文件：以某种编码存储的二进制序列，并可以解码为人可以阅读的文件，如配置文件、源代码文件等
    • 二进制文件：纯01，无编码/解码，有相应软件解释，如图像文件、音频文件等

  • 常见字符编码：

    • ASCII码：美国信息交换标注码(American Standad Code for Information Interchange), 7位，128字符
    • GB2312\GBK\GB18030:中文码，六千多/两位多/七万多，向前兼容
    • BIG5：繁体中文码
    • UNICODE码：
      • UTF-8编码：1到4字节，变长
      • UTF-16：2或4字节，相当于定长

  • Python字符串的编码和解码：encode函数和decode函数

    文件读写
    读写基础

    文件读写一般步骤：

    • 打开并创建文件对象

    • 读、写等操作

    • 关闭并释放文件对象

    • 绝对路径和相对路径

      • 绝对路径：文件所在位置的完整路径，总是从根目录开始

      • 相对路径：文件相对于当前工作目录所在的位置

        • “./”:当前目录

        • “…/”:当前目录的父目录

    • 文件打开模式：

  • 打开文件：内置函数open(filename, mode=“r”,encoding=None…)

    • filename:相对或绝对文件名，推荐相对
    • mode：打开模式，默认为只读和文本模式
    • encoding：编码，默认为本机编码

    关闭文件：内置函数close()

    I/O重新定向

    • 标准输入：input函数默认接受键盘输入，可修改为文件对象

    • 标准输出：print函数默认输出至屏幕，可修改为文件对象

写文件

  • write(content):写入字符串，无换行符

  • writelines(iterstr):将字符串的可迭代对象写入文件

读文件
  • read(size=-1): 默认返回文件所有内容的字符串。可通过size指定都多少字符（字节）
  • readlines(size=-1):按行读
  • readlines(hint=-1):返回内容行集
辅助功能

  • tell():返回当前位置，单位是字节

  • seek(offset, from):移动位置

    • offset:偏移量，字节数
    • from:0,表示文件头；1标识当前位置表示文件尾。字节模式下没问题，字符模式下from为1or2时，offset只能是0
